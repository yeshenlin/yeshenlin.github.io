Java深拷贝

Java中浅拷贝和深拷贝的区别：浅拷贝对类对象一无所知，如果对象中包含的所有数据类型都是基本类型或基本类型，拷贝没什么问题。但是对象中包含类对象，拷贝的结果使两个域引用同一个对象。

  1. 实现Cloneable接口
