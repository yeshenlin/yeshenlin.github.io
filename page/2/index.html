LeetCode 120. Triangle

The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

动态规划的本质就是记忆化搜索，把上次计算的东西保存下来，可以重复利用。不用每次都从头到尾走一遍，因此动态规划比搜索要快。
Advantage： Easy to think and implement.
Disadvantage: Expensice memory cost.

triangle的两种解法：
解法一：从上往下

解法二：从下往上

Java多线程

进程与线程的区别：
进程： 每一个进程都有独立的代码和数据空间(进程上下文)，进程间的切换有较大的开销，一个进程包含1-n个线程。进程是系统资源分配的最小单位。
线程： 线程相对于进程更加轻量，上下文信息更少，创建和销毁更加简单。多个线程共享一个进程的资源，OS的许多资源分配和管理都是进程级别的，线程是OS调度的最小单位。

I/O密集性的场景适合使用多进程，计算密集型的场景适合多线程。

这可怎么办？

上海的秋天越来越深了，气温降得很快，还带着冰凉的秋雨。风也越来越大了，早上起床站在阳台上看到两只小猫在食堂侧门墙角缩着。每年冬天学校的小猫被冻死的可能性很大，眼看着冬天就要来了，这该怎么办？

学校有一些很善良的人，在每天路过食堂会把手头上的食物分给小猫一点，但是他们应该不会把小猫带回宿舍去养的。因为学校有学校的规章制度，不准学生在宿舍里养小动物。

想起研一的冬天在路边捡的小叶子，带回宿舍跟它朝夕相处，每天我吃饭的时候，它也在吃饭；我看书的时候，它趴在我手边睡觉；我去上课，它就躲在我抽屉里睡觉。无论是猫还是狗，是名贵物种还是户外流浪的，这些都是生命。小叶子很聪明，刚抱回来巴掌那么大，喜欢趴在我手掌里睡觉。两三个月的时间，就长挺大了，从地面跳到书桌上完全跟玩儿似的。它不爱洗澡，每次给它洗澡它都要甩我一身水，叫声凄惨:P

可惜，后来把小叶子送回母猫身边了。后来我也搬了校区了，不知道它现在是不是天天在闵行校区的草丛里嘻嘻打闹、爬树翻墙。很庆幸有这样一只可爱的精灵陪伴我的那段时光，我总是在看到其它动物的时候想起它。猫的记忆应该像个小孩子一样吧，不知道它还记不记得我。

在这个校区里，我再也不是一个人一间宿舍了，不能把食堂门口的两只小猫带回宿舍了。希望它们能度过这个冬天。

Dict的特点：

  1. 查找和插入的速度极快，不对key值增加而增加；
  2. 需要占用大量内存，内存浪费大。

list的特点：

  1. 查找和插入的时间随着元素的增加而增加；
  2. 占用空间小，浪费内存很少。

注意：由于Dict的底层原理是Hash算法，为了保证hash的正确性，Dict的key值必须是不可变类型，Python中的str、int、tuple是不可变的，list是可变的。因此使用list做key会报错。
